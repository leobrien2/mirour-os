For Deep Dive Architecture & Technical State
CORE PROBLEM WHY EVERYTHING BLOCKED
The problem: Product data from brands is messy. CSVs have no images, no tags, minimal descriptions. Web scraping worked for demo but isn't scalable.

The solution: LLM-powered auto-tagging. Send onboarding context + 100 products → LLM returns consistent tags → store in DB. Repeat per batch.

Why this is IP: Two-way sync between Mirour tags and ecom platforms means brands' stores improve just by using Mirour. Tags flow back.

WORKING

Flow Builder

Node graph editor with 7 step types. Conditional logic, tag assignment, recommendations. Side-by-side preview shipped Mar 20.

WORKING

Flow Runner

Customer-facing quiz at /f/[formId]. Renders steps, calculates tags from answers, filters products by tag match.

WORKING

Tag-Based Recommendations

3 strategies: "any" match, "all" match, "zone-first". Quiz answers accumulate tags → filterProducts() finds matches.

NOT WORKING

Product Import Pipeline

No automated import. Brands export CSV → manually processes → sits in CSV never imported to Supabase. Store integrations UI saves keys but NO code calls APIs.

GHOST TABLE

Analytics Events

analytics_events table created. Zero code writes to it. Can't measure funnel, conversion, drop-off.

GHOST TABLE

Customer Tags

customer_tags table exists. appendTags() and getSessionTags() ready. Flow runner NEVER calls them.

GHOST TABLE

Form Visits / Flow Sessions

form_visits and flow_sessions tables exist. Neither written to. Can't measure engagement or completion rates.

PARTIALLY BUILT

Store Integrations

Users save Lightspeed, Shopify, Squarespace keys. Keys stored. But NO code calls those APIs to sync products.

OVERBUILT

Zone System

Maps products to physical store areas. No pilot uses zones. Adds cognitive overhead with zero current value. Hide for now.

FUTURE

AI Flow Generation

Import products → AI reads categories/descriptions → auto-generates quiz flow. Eliminates blank canvas. #1 simplification for onboarding.

FUTURE

AI Auto-Tagging

LLM reads product name + description + category → generates tags. Turns week-long onboarding into same-day.

FUTURE

Customer Insights Dashboard

Once analytics + customer tags work: "62% want non-alcoholic" / "Top vibe: calm evening". Makes $199/mo feel like a steal.
